Logic and Proofs

Logic is key to a computer scientists work. It guides the language and instruction as the scientist interfaces with the computer. It is important to build a stron foundation in logic and proofing as these principles will be used constantly.

Concepts introduced in this section are propositions, truth values, truth tables, logical operations, and compound statements

Logic is a study of reasoning, of course logic has to start with something. The most basic type of logic is a proposition a statement that is true or false. An example of this is "The cat has four legs." this simple statement can be either true or false, yes or no, on or off, or in computer terms 0 or 1.
